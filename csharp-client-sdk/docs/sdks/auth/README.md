* [ApiKeyAuth](#apikeyauth)

## ApiKeyAuth

### Example Usage

```csharp
using Openapi;
using Openapi.Models.Operations;

var sdk = new SDK(
    globalPathParam: 100,
    globalQueryParam: "some example global query param"
);

var res = await sdk.Auth.ApiKeyAuthAsync(new ApiKeyAuthSecurity() {
    ApiKeyAuth = "Token YOUR_API_KEY",
});

// handle response
```
